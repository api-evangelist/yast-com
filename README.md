# Yast.comThis is a repo for managing machine readable artifacts for the [Yast.com APIs](http://www.yast.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Yast.com APIs](http://www.yast.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.yast.com)- [Developer](http://www.yast.com)- [Twitter](https://twitter.com/Yastdotcom)- [GitHub](https://github.com/yast)The [APIs.json](https://github.com/api-evangelist/yast-com/blob/master/apis.json) index provides links to all the relevant links for the Yast.com APIs, including the OpenAPI and Postman if they are present.
